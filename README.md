## Travel Notes
SPA-приложение для хранения заметок о путешествиях с авторизацией и работой с API.

## Возможности
- Авторизация пользователя
- Просмотр списка путешествий
- Добавление записи
- Редактирование записи
- Удаление записи
- Валидация формы
- Защита приватных маршрутов

## Используемые технологии
- React 18
- TypeScript
- Vite
- React Router
- Axios
- TanStack Query
- Zustand
- Yup
- React Hook Form

## Запуск проекта
```bash
npm install
npm run dev
```
